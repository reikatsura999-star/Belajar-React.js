# APLIKASI TODO LIST

## Deskripsi Proyek

Aplikasi To-Do List Sederhana ini dirancang khusus untuk membantu Anda mengatur kegiatan dengan lebih mudah, cepat, dan teratur. Dengan tampilan yang bersih dan fitur yang sederhana, aplikasi ini sangat cocok untuk Anda yang ingin menyelesaikan pekerjaan tanpa ribet.

## Panduan Penggunaan Aplikasi

### 1. Menambahkan Tugas
Pengguna cukup memasukkan tugas yang ingin ditambahkan lalu klik "Tambah". Secara otomatis, data akan tersimpan dalam daftar.

### 2. Melihat Daftar Tugas

- **All**: Menampilkan semua daftar tugas yang tersimpan.
- **Active**: Menampilkan daftar tugas yang belum selesai.
- **Completed**: Menampilkan daftar tugas yang sudah diselesaikan.

### 3. Menandai Tugas Selesai
Untuk menandai tugas sebagai selesai, cukup centang kotak checkbox di samping teks. Tugas tersebut otomatis akan berpindah ke daftar Completed.

### 4. Menghapus Tugas
Setelah tugas tidak lagi diperlukan, pengguna dapat menghapusnya dengan klik tanda "X" di sisi kanan teks.

## Fitur yang Sudah Diterapkan

1. Menambah tugas baru dengan cepat dan otomatis tersimpan.
2. Melihat semua tugas, tugas aktif, dan tugas yang sudah selesai.
3. Menandai tugas sebagai selesai.
4. Menghapus tugas.

## Catatan Tambahan

Aplikasi ini dibuat menggunakan library React dengan struktur yang sederhana. Terdapat 4 file JSX sebagai komponen utama dan 1 file utama yang berfungsi sebagai tempat untuk melakukan import. Selain itu, aplikasi ini juga menggunakan 2 file CSS untuk mengatur tampilan.
